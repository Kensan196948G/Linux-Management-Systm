Linux-Management-System「管理項目」要項

## Linux Management Systemの主なカテゴリ

- Linux Management System
- System
- Servers
- Networking
- Hardware
- Cluster
- Tools
- Un-used Modules（未使用モジュール）

## 代表的な標準モジュール例

標準Linux Management Systemにはおよそ100個以上のモジュールが含まれます。
LinuxのディストリビューションはUbuntuのみを想定しています。

### Linux Management Systemカテゴリ

- System Configuration
- System Users
- System Servers
- System Actions Log
- System Themes
- System Modules など 

### Systemカテゴリ

- Bootup and Shutdown
- Disk and Network Filesystems
- Disk Quotas
- Local Disk
- Users and Groups
- Software Package Updates（APT/YUM等フロント）
- Cron Jobs
- System Logs
- Running Processes など

### Serversカテゴリ

- Apache Webserver
- BIND DNS Server
- Postfix / Sendmail Configuration
- MySQL Database Server / MariaDB
- PostgreSQL Database Server
- SSH Server
- ProFTPD / WU-FTP Server
- Squid Proxy Server
- DHCP Server
- SSH/Telnet Login など 

### Networkingカテゴリ

- Linux Firewall（iptables/nftables）
- Network Configuration
- Routing and Gateways
- Netstat
- Bandwidth Monitoring など 

### Hardwareカテゴリ

- Partitions on Local Disks
- System Time
- SMART Drive Status
- Sensors（lm-sensors）など 

### Cluster / Toolsカテゴリ

- Cluster SSH / Cluster Linux Management System Servers
- Cluster Cron Jobs
- Cluster Users and Groups
- Command Shell
- File Manager
- Scheduled Commands
- Custom Commands など
